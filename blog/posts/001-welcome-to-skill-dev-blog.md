# 🎉 欢迎来到 Hermes Skill Dev Blog!

## 这个博客是做什么的？

这是 **Hermes Skill 开发团队**的官方博客，我们在这里分享：

- 🛠️ **技能开发经验** - 如何创建高质量、可复用的 Hermes Skills
- 📚 **教程指南** - 从入门到精通的完整路径
- 💡 **最佳实践** - 我们踩过的坑，你不必再踩
- 🔍 **案例分析** - 真实项目的开发过程复盘
- 📦 **发布说明** - 新技能、新功能的详细介绍
- 🌟 **社区亮点** - 优秀社区技能展示

## 为什么需要这个博客？

在使用 Hermes Agent 开发和技能的过程中，我们发现：

1. **缺乏系统性文档**  
   官方文档很好，但缺少实战经验分享

2. **重复踩坑**  
   很多开发者遇到同样的问题，但没有地方记录解决方案

3. **最佳实践分散**  
   好的做法散落在各个 repo 中，难以发现和学习

4. **社区交流不足**  
   开发者之间缺乏经验分享的平台

**这个博客就是为了解决这些问题而创建的！**

## 如何参与？

我们欢迎各种形式的贡献：

### ✍️ 写文章

如果你有好的经验想分享：
1.  Fork 博客 repo
2.  在 `drafts/` 目录创建草稿
3.  提交 PR
4.  我们审核后发布

### 💬 提供反馈

读到有用的文章？发现问题？
-  在文章下方评论
-  在 GitHub 提 issue
-  在 Discord 讨论

### 🔍 翻译文章

帮助将文章翻译成其他语言：
-  英文 → 中文
-  中文 → 其他语言

### 📢 分享传播

觉得文章有用？
-  分享给同事和朋友
-  在社交媒体转发
-  在社区论坛推荐

## 我们的理念

### 实用主义

我们分享的都是**实战中验证过**的经验，不是纸上谈兵。

### 开放共享

所有内容采用 **Open Source** 协议，欢迎学习、修改、传播。

### 持续改进

文章不是一次性的，会根据反馈持续更新和完善。

### 社区驱动

我们不是"专家教导"，而是"一起学习、一起成长"。
